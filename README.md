# Workout Tracker

A command-line workout tracking application built in Python as the Final Project for Harvard's CS50P course.

## Features
- Add workouts (exercise, reps, sets, weight)
- View all workouts in a formatted table
- Delete workouts by index
- Data stored in CSV file with automatic date

## How to Run
python project.py

## Testing
pytest test_project.py

## Files
- `project.py` — Main program
- `test_project.py` — Unit tests
- `workout.csv` — Auto generated data file
